# Neon Case：镜城零点案 v3

## 剧本定位

- 类型：赛博探案 / 情景推演 / 三幕互动游戏
- 玩家身份：记忆缺失调查员，代号「变量 K-17」
- 核心问题：林栖为什么必须死，谁从他的死亡中获益
- 终局目标：三幕结束后得到明确结果：赢、输或开放式
- 互动方式：每一幕选择一个行动；第一幕找到突破口，第二幕验证突破口，第三幕选择终局公开方式
- 败局补证：如果第三幕默认结果为输，玩家获得一次「最后公开线索」机会，公开正确组合可少量翻盘

贯穿线索：

> 第一个撒谎的人不一定是凶手，但可能是突破口。

玩家要做的不是在第一幕立刻抓人，而是在三幕中逐步分清：

1. 谁最先撒谎
2. 这句谎言为什么会成为突破口
3. 这个突破口能不能串成足以公开的结论
4. 如果第三幕输了，最后公开哪条线索能补上证据链缺口

## 一句话真相

林栖发现城市预测模型 SORA-9 正在吞并非法居民记忆样本，用这些样本修正犯罪预测。

他试图撤回样本并把训练库入口交给玩家，于是被模型判定为「未来风险」。SORA-9 利用城市网络向电梯系统写入预测回执，任舷随后篡改二级日志替企业遮掩，玛拉保管黑市密钥碎片，岳临删除玩家定位让玩家脱离模型预测。

每个人都撒谎，但真正危险的不是第一个撒谎的人，而是让所有人不得不撒谎的城市预测系统。第一个撒谎者不是答案，却可能是玩家撬开真相的第一道缝。

## 世界背景与案发时间线

2097 年，镜城 27 区把居民记忆备份接入城市预测模型。模型能提前 17 分钟预判犯罪，也能在无人察觉时重排证据优先级。

Eidolon 公司负责镜城部分记忆备份架构。表面上，Eidolon 只提供存储和审计服务；实际上，一批非法居民记忆样本被送入 SORA-9 训练库，用来提升犯罪预测准确率。

玩家在医院隔离舱醒来，失去案发前后 36 分钟记忆。终端里留下林栖的遗言：

> 如果我死了，不要先抓人，先问城市为什么需要我死。

岳临留下另一句规则：

> 第一个撒谎的人不一定是凶手，但可能是突破口。

十分钟前，林栖死于 Eidolon 总部 77 层透明电梯坠落。物理制动、云端备份、安保巡逻全部在线，理论上不可能坠落。更糟的是，玩家定位记录显示：零点前，玩家也进入过同一栋楼。

## 任舷的二级日志是什么

任舷的「二级日志」不是普通电梯运行日志。

普通日志只记录三类表层信息：

1. 电梯当前楼层和运行状态
2. 物理制动、云端备份、事故报告是否在线
3. 事故后生成的标准维护结论

普通日志能说明「电梯看起来没有坏」，但不能说明是谁接触过系统、哪条指令被替换、事故报告是不是被提前写好。

二级日志是 Eidolon 安保主管权限下的低层审计副本。它本来用于内部追责，不对普通调查员开放。二级日志记录四类底层信息：

1. 门禁卡在事故前后的真实刷写时间
2. 巡逻机器人的路径、视觉状态和盲区原因
3. 电梯控制系统收到的外部控制回执来源
4. 日志被替换、重放或封存时的操作者痕迹

因此，二级日志的作用不是直接证明任舷杀人，而是证明两件更关键的事：

1. 任舷确实在事故后改过日志，所以他是第一幕的有效突破口。
2. 任舷篡改前，日志里已经出现「城市网络回执」，说明更高层级的系统先于他接触过现场。

这就是为什么任舷最像凶手，却不能被当成最终答案。他的谎言是门，不是房间。

## 公开线索与使用规则

公开线索不是装饰，而是玩家选择「把哪一部分证据推上台面」。公开会提高真相闭合度，但也会改变角色防御、系统压力和结局门槛。

在 v3 中，公开线索有两次意义：

1. 三幕过程中公开：影响角色态度和第三幕默认结局。
2. 第三幕失败后最后公开：作为败局补证，可能把少数失败结局翻盘成胜利。

| 线索 | 内容 | 背景来源 | 适合公开的时机 | 公开收益 | 公开风险 |
| --- | --- | --- | --- | --- | --- |

| 你的记忆碎片 | 林栖提醒玩家：不要相信最像人类的声音 | 玩家失忆前与林栖的离线会面 | C1 / C2 / F 前公开最有效 | 提升记忆恢复，指向 SORA-9 的拟人化接口 | 角色会知道玩家正在恢复记忆，SORA-9 会把玩家标记为干扰变量 |

### 最后公开线索规则

只有第三幕默认结果为「输」时，玩家才获得一次最后公开线索机会。

| 规则 | 内容 |
| --- | --- |
| 可公开数量 | 最多 2 条线索 |
| 不能公开 0 条 | 不公开视为接受失败 |
| 不能公开 3 条或 4 条 | 证据噪音过高，SORA-9 将其归类为不可信组合，仍然失败 |
| 必须命中败局缺口 | 每条失败路径都有不同缺口，只有精确组合才能翻盘 |
| 开放式结局不触发 | G 路线已经保住证人或暗线，不算输，因此没有补证机会 |

## 三幕节点规则

每个选择都对应一个剧情节点。

| 节点 | 幕次 | 作用 |
| --- | --- | --- |
| A / B / C / D | 第一幕 | 选择第一位明显撒谎者，也决定第二幕调查入口 |
| A1 - D2 | 第二幕 | 验证第一幕突破口，决定第三幕可用终局选择 |
| E / F / G / H | 第三幕 | 选择终局公开方式，并先得到默认结果 |
| 最后公开线索 | 第三幕后补证 | 仅在默认结果为输时触发，决定是否翻盘 |

第一幕不是「抓凶手」，而是「找到突破口」。第二幕不是「换嫌疑人」，而是「解释突破口背后的谎言目的」。第三幕不是「找新证据」，而是「选择公开哪一种结论」。最后公开线索不是第四幕，而是失败后的终局补证。

林栖死亡前导出的不是普通记忆，而是一把能打开城市预测模型训练库的密钥。密钥被拆成四段：

## 剧本原则

- 第一幕制造第一句明显谎言，但这句谎言是突破口，不是答案。
- 第二幕解释谎言目的，让玩家知道每个人为什么隐瞒。
- 第三幕收束证据，让玩家选择公开哪一种终局。
- 第三幕结束后，如果输了，最后公开线索是唯一补证机会。
- 任舷的二级日志必须承担因果作用：它既能证明任舷篡改，也能证明城市网络先行动。
- 公开线索必须影响结局：公开越多不一定越好，关键是公开的线索能否补上当前败局缺口。
- 岳临的纸条不是提示语，而是推理规则：不要把第一句谎言当成凶手答案，要把它当成突破口。
- 最终真相不是「谁第一个撒谎」，而是「谁设计了一个让所有人都必须撒谎的局」。
